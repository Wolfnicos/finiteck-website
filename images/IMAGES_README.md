# Image Assets

Replace these placeholder files with actual assets:

## Required Images

### app-icon.png (or use app-icon.svg)
- Size: 512x512 pixels (or 1024x1024 for retina)
- Format: PNG with transparency
- Used: Hero section, footer
- Note: Use the same icon as your iOS app

### app-store-badge.svg
- Current file is a placeholder
- Download official badge from: https://developer.apple.com/app-store/marketing/guidelines/
- Use the black version for dark backgrounds

## Optional Images

### screenshot-1.png
- Size: iPhone screenshot dimensions (e.g., 1290x2796 for iPhone 15 Pro Max)
- Format: PNG
- Used: Can be added to hero section for visual appeal

## Notes

- Keep file sizes optimized for web
- Use descriptive alt text in HTML for accessibility
- SVG is preferred for icons and badges (scalable, small file size)
